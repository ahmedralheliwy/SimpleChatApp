        <li>Run migrations: <code>python manage.py migrate</code> ⚙️</li>
        <li>Start the server: <code>python manage.py runserver</code> 🌟</li>
        <li>Open in your browser: <a href="http://127.0.0.1:8000">http://127.0.0.1:8000</a> 🌐</li>
